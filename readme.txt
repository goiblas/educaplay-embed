== Description ==

Embed Educaplay activities in your WordPress site

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/embed-activities-for-educaplay` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
